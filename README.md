AI Coach Video Call App
An advanced, AI-powered video conferencing platform designed for interactive coaching sessions. This application uses modern web technologies to provide a video calling experience integrated with AI capabilities

Features
Real-Time Video Communication: High-quality video and audio streaming for coaching sessions.
AI Integration: Built-in AI coaching logic to enhance user interaction.
Modern UI/UX: A clean, responsive interface built with Next.js and Tailwind CSS
.
Type Safety: Developed with TypeScript for robust and maintainable code (comprising 98.5% of the codebase)
.
Database Management: Integrated with Drizzle ORM for efficient data handling
.
Tech Stack
Framework: Next.js (App Router)
Language: TypeScript
Database/ORM: PostgreSQL, Drizzle ORM
Styling: Tailwind CSS & PostCSS
UI Components: Styled using modern component architectures
.
